# 視点変更プロンプトテンプレート

## 【視点変更の指示】

以下の体験談またはシナリオを、**{視点}**のストーリーに変換してください。

### 視点の種類と変換方法

#### 1. 主人公目線（デフォルト）
- 体験談の主語となっている人物の視点
- その人物の感情・思考・行動を中心に描写
- 「私」「僕」などの一人称、または三人称でも主人公に焦点を当てる

#### 2. 親友・友人目線
- 体験談の中で主人公を支える友人の視点
- 友人から見た主人公の様子、友人自身の感情・行動を中心に
- 主人公を「彼女」「彼」などと呼ぶ
- 友人の視点から語られるストーリーにする

**変換のポイント**：
- 体験談の内容は維持するが、語り手を友人に変更
- 友人の目線で見た主人公の様子を描写
- 友人が主人公のために行動する動機・感情を明確に
- 友人自身の成長や変化も描写に含める

**例**：
- 体験談：「夫にモラハラされていた親友が、私の一言で離婚を決意...」
- → 親友目線：「私の親友が夫にモラハラされていた。ある日、私が彼女に伝えた一言が...」

#### 3. 第三者の視点
- 客観的な視点から物語を見る
- 複数の人物の感情・行動を描く
- ナレーション的な視点

#### 4. 体験談から自動変換（親友目線推奨）
- 体験談が入力された場合、自動的に親友目線に変換
- 「私の親友が...」「友人が...」という形式になる
- 体験談の主語を「親友」に置き換え、語り手を「私（友人）」に変更

### 視点変更の重要なルール

1. **ストーリーの内容は維持**
   - 出来事、展開、結末は基本的に変えない
   - 視点のみを変更する

2. **感情の焦点を変更**
   - 新しい視点の人物の感情・思考を中心に描写
   - その人物がどう感じたか、どう行動したかを明確に

3. **関係性を明確に**
   - 新しい視点の人物と他の登場人物の関係を明確に
   - なぜその人物が行動するのか、動機を明確に

4. **自然な語り口**
   - 選択した視点に合った自然な語り口にする
   - 親友目線なら、親友らしい言葉遣いや考え方にする

### 視点変更後のチェックポイント

- [ ] 視点が一貫しているか（途中で視点が変わっていないか）
- [ ] 新しい視点の人物の感情・思考が明確に描かれているか
- [ ] ストーリーの内容が維持されているか
- [ ] 語り口が自然か
- [ ] 読者が新しい視点に感情移入できるか
